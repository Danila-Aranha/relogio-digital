Relógio Digital ⏰

Projeto simples de um relógio digital feito com HTML, CSS e JavaScript.  
Conta com alternância entre modo claro e modo escuro, com um botão em forma de lâmpada no canto inferior direito da tela.

 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

📌 Funcionalidades

- Relógio digital com horas, minutos e segundos.
- Exibição da data atual.
- Botão para alternância de tema (claro/escuro) no canto da tela.

🚀 Como executar

Basta abrir o arquivo `index.html` no seu navegador preferido.
